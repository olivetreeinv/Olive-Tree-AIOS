## Key Takeaways
1. Lock up off-market/owner-financed deals with contingencies tied to seller completion timelines rather than standard 30-day periods—use 'earnest money fully refundable' + extended escrow tied to renovation completion
2. For residential FHA deals, contingencies must be removed in writing (not automatic like commercial); use 'notice to perform' language to extend timelines if renovations incomplete
3. Owner-financed deals eliminate underwriting friction: request bank statements directly from mom-and-pop sellers to build P&L yourself; almost 50% of Brennan's deals use owner financing
4. Self-manage small units (4-6 unit) using Lorex wired cameras ($300-400 system), one maintenance person on ground, boots-on-the-ground emergency contact, and Rent Tech software (~$50/month) instead of 8% PM fee
5. For laundry/washer-dryer operations, lease equipment via CSC or similar (24-36 month agreements) rather than buying; read cancellation clauses carefully to avoid auto-renewal traps
6. Mobile home parks: target <100 pads (Brennan owns 6 smaller parks clustered together, cross-collateralized as single loan), ensure surrounding single-family homes >$100k value + apartments >$1,000/month rent + properties within 5 miles of Walmart
7. Post-closing deed structure: close residential deals in personal name, then quitclaim to LLC within 7 days via escrow without lender notification; establish LLC in any state (Wyoming preferred for cost at $100), not required to match property state for residential
