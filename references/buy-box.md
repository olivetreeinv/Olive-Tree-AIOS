## Universal Criteria — Apply to All Markets

These apply regardless of market. A deal that misses any of these is a pass before you open the OM.

| Criterion | Requirement |
|---|---|
| **Unit count** | 15–50 preferred · analysis floor 5+ |
| **Purchase price** | $1M–$3M |
| **Property type** | Multifamily (apartment buildings, townhome-style communities) |
| **Strategy** | Value-add or stabilized with operational upside — not fully priced stabilized assets |
| **Seller motivation** | Must have upside available: below-market rents, operational inefficiency, deferred maintenance, or off-market pricing |
| **Hold period** | 4–6 years |
| **Return target** | 18.21% annual ROI, 2.09x equity multiple |
| **Preferred return** | 6% pref, then 70/30 LP/GP split |
| **Min investment** | $25K (individual LPs), accredited and non-accredited |
| **Deal sourcing** | On-market and off-market both acceptable |

## Market Profiles

### 1 — Chamblee, GA (30341)
**Strategy:** Value-add
**Vintage target:** 1965–1990
**What I'm looking for:**
- Rents below market — room to push through renovations and better management
- Assets near MARTA stations or major corridors (Peachtree Industrial, Chamblee Tucker Rd)
- Working professional tenant base
- Classic interiors with renovation upside

**Pricing:** $90K–$140K/unit depending on condition
**Move speed:** Fast — can move quickly on deals with clear upside
**Red flags:** Fully renovated / fully priced, no MARTA proximity, heavy deferred maintenance with no upside story

## How to Use This Doc

**In market research (`/market-research`):**
When a deal comes in, check this doc first. Confirm the zip code is in an active market. If it's not listed here, ask Brian before running research.

**In underwriting (Underwriting Skill — coming):**
Use the per-market price/unit target as a sanity check on purchase price. Use the vintage target to frame renovation scope. Use the strategy type (value-add vs. stabilized) to calibrate year-1 underwriting assumptions.

**In broker outreach:**
When prospecting, lead with the specific zip codes from this list. Brokers respond better to specific asks ("I'm looking for 20–40 units in 30341, 1970s vintage, value-add") than generic requests.

**In LP conversations:**
The market profiles explain the investment thesis per geography. Use them when explaining why you're looking in a particular market.
