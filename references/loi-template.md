# LOI Template — Quick Reference

Full template (legal text): `templates/loi-template.md` — read that when drafting.

---

## Standard Defaults

```
Financing:    Bridge loan, non-recourse, 70–75% LTC, 36-month I/O
              1% lender fee, 0.5% broker fee, 6-month extension option
Deposit:      1–2% of offer price total | 50% hard from day 1
DD Period:    28–30 days (ask for 45 on complex deals)
Closing:      60 days from LOI execution
Extension:    30 days / $1,300 (adjust to ~0.1% of purchase price)
Commissions:  Seller pays — always
Contingency:  No financing contingency
```

## Field Guide

| Placeholder | What to fill | Default |
|---|---|---|
| `[BROKER_NAME]` | Broker's full name | From Brokers List |
| `[BROKERAGE]` | Brokerage firm | From Brokers List |
| `[PROPERTY_ADDRESS]` | Full address | From OM or listing |
| `[OFFER_PRICE]` | Offer amount | From deal analysis |
| `[PRICE_PER_UNIT]` | Offer ÷ units | Auto-calc |
| `[TOTAL_DEPOSIT]` | Total earnest money | ~1% of offer |
| `[HARD_DEPOSIT]` | Hard day-1 portion | ~50% of deposit |
| `[FINANCING_TYPE]` | Loan type + LTV | `70% LTC Bridge Financing` |
| `[DOWNPAYMENT_PCT]` | Equity % | `30` |
| `[EQUITY_SOURCE]` | Equity origin | `equity sourced from our funds & partners` |
| `[DD_DAYS]` | DD period | `28` |
| `[CLOSING_DAYS]` | Closing timeline | `60` |
| `[EXTENSION_DAYS]` | Extension window | `30` |
| `[EXTENSION_DEPOSIT]` | Extension deposit | `$1,300` |
| `[SPECIAL_CONDITIONS]` | Deal-specific terms | `N/A` |
| `[DATE]` | Submission date | Today |

## Negotiation Targets

- **Price** — anchor below asking, leave room to negotiate up
- **Deposit** — keep hard portion low, push for longer soft period
- **DD period** — 30+ days preferred; 45 on complex deals
- **Closing** — 60 days standard; extend for complex deals
- Ask broker: *"What terms matter most to the seller?"*
